# Python File Organizer Automation

This Python script automatically organizes files in a folder based on file type.

## Features
- Organizes files by extension
- Safe dry-run mode (default)
- Simple command-line usage
- No external dependencies

## Use cases
- Organizing Downloads folder
- Cleaning messy directories
- Bulk file management
- Automation tasks

## How to run

### Dry run (recommended)
```bash
python organizer.py sample_folder
```
